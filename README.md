# timeWebS3
Компонент загрузки файлов и выгрузки объектов в timeWeb для Delphi XE10.1 и выше

Используются компоненты Indy, встроенные в Berlin версию.
Основные модули: сам компонент + модуль потоков 

В список добавлены: простой класс обработки ошибок (с логированием и списокм), файлы логирования и диалог сообщений
